Go 静态代码检查

发表于2023-03-23|更新于2023-03-25
|字数总计:374|阅读时长:1分钟

GolangCI-Lint 是一个 lint 聚合器:

  • ⚡ 非常快：并行运行 linters，重复使用 Go build cache 和缓存分析结果。
  • ⚙️ 基于 Yaml 的配置。
  • 🖥 与 VS Code、Sublime Text、GoLand、GNU Emacs、Vim、Atom、GitHub Actions 集成。
  • 🥇 包含了大量的翻译器，不需要安装。
  • 📈 由于调整了默认设置，假阳性的数量最少。
  • 🔥 带有颜色、源代码行和标记标识符的漂亮输出。
